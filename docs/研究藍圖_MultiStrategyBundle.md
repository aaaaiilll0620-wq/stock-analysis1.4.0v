# 研究藍圖 — 時空分流策略組合（Multi-Strategy Bundle）

**定稿日：2026-07-18。** 本文件取代「單一 C2 通用核心」的隱含路線，改採
「主系統（資金分配器）＋ 短/中/長線三個解耦子模組」架構。它是 backlog 的排序依據，
不是預註冊文件——每個 R 項開工前仍須各自預註冊（單發射擊制照舊）。

## 0. 架構決策與其邊界

- **決策**：策略層解耦為 A（短線）/ B（中線）/ C（長線）三個子模組，各自適配該時間
  尺度的底層 alpha；資金比例由主系統決定。
- **C2 不廢除、改編制**：C2 只在 20 日窗/月度級訊號/季度執行的尺度上被驗證與宣稱有效
  （六時代 hold-out、ledger 活體 IC +0.054、季度淨 +18.6%/年），它直接就位為**模組 B 現任引擎**。
  汰換 B 引擎的唯一途徑＝新候選過同一道閘門且勝出，架構重組本身不構成汰換理由。
- **既有負結果約束模組 A 的假設**：寬池 20 日動能反轉（§19 alpha gate）、短線動能濾網
  未採用（§18）。模組 A 的預設假設是「扣成本後可行的短線 alpha 可能不存在」，
  由數據反駁；閘門過不了就正式結案「短線不做」，負結果照登 devlog。

## 1. 使用者執行約束（模組 A 成本模型依據，2026-07-18 確認）

- 券商：元大證券，網路下單 **6 折** → 單邊手續費 **0.0855%**；證交稅賣出 **0.3%**（不當沖，無減半）。
- 來回成本 **0.471%**（買 0.0855% ／ 賣 0.3855%）。整股舊模型 0.617%（§22 用），
  新實驗一律改用本費率；§22 已結案判定不回頭改，僅在 R3 敏感度重估月/季 drag。
- **以零股為主**（學生資金規模，不融資、不當沖）。零股最低手續費 1 元 →
  單筆 ≥ 約 1,200 元即不觸發最低費；部位規模每檔數千元起即無感。
- 零股特有風險（回測模型外，需活體實測）：分盤集合競價撮合、冷門股零股可能**無成交**、
  零股成交價 vs 整股日線的溢折價**無歷史資料可回測** → R3 凍結滑價敏感度網格
  （0/10/20/30bps），並用演練 fills 基建以小額真實零股單實測分布回填。
