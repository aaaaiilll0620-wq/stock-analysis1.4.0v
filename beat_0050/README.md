# beat_0050 — 做出「淨贏 0050」的選股策略計畫

> 獨立計畫資料夾。目標唯一:設計一個**樣本外、含息、扣成本後淨夏普 > 0050 買進持有**的選股策略。
> 這裡的一切都以 0050 含息買進持有為唯一基準,**不准再用等權除息母體自嗨**(上一輪翻車教訓)。

## 為什麼從頭來
原綜合分/雙確認策略經誠實回測(含息+成本+對 0050)後**沒贏過 0050**。
實測:dual-confirm 含息後 2019-26 CAGR 22.8% / 夏普 1.05,但同窗 0050 是 31.1% / 1.31 → 仍輸。
使用者決定重來,認真設計能贏大盤的策略,從「先建誠實基礎」開始。

## 資料夾結構

## 地基:honest_backtest.Engine
餵 `holdings={as_of:[stock_id]}` → 回傳:
- **含息**:個股價格報酬(obs_alpha.fwd) + dividend_yield_TSE/12 (duckdb ASOF join)
- **扣成本**:元大 6 折來回 0.47% + 滑價 0.10%,逐月週轉率計費
- **對 0050**:市值加權含息買進持有為基準
- **六時代**:2005-2009 海嘯 / 2010-2014 / 2015-2018 / 2019-2021 / 2022 空頭 / 2023-2026
- **風險優先**:CAGR / 夏普 / Sortino / MDD / 水下月數 / 勝率

用法:
```python
from beat_0050.honest_backtest import Engine
eng = Engine()
result = eng.run(holdings_by_asof)     # {as_of: [stock_id,...]}
eng.report(result, "我的策略")          # 印指標表 + 對 0050 判定
```

4. **紙上實盤** → 再碰真錢。
